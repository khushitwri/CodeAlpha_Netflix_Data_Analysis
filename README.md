# CodeAlpha Data Analytics Internship Projects

## Project Overview

This repository contains multiple data analytics projects completed as part of the CodeAlpha Data Analytics Internship Program. The projects focus on data analysis, visualization, and sentiment analysis using Python and its libraries.

## Tools and Libraries

* Python
* Pandas
* Matplotlib

## Project 1: Netflix Data Analysis

### Objective

Analyze Netflix Movies and TV Shows data to identify content trends and insights.

### Analysis Performed

* Release trend analysis over the years
* Movies vs TV Shows distribution
* Top 10 countries with highest Netflix content

### Key Findings

* Netflix content increased significantly after 2015.
* Movies constitute a larger portion of the catalog than TV Shows.
* A small number of countries contribute a major share of Netflix content.

---

## Project 2: Sentiment Analysis

### Objective

Analyze customer reviews and classify sentiments based on ratings.

### Analysis Performed

* Positive and Negative sentiment classification
* Sentiment distribution analysis
* Rating distribution visualization

### Key Findings

* Positive reviews dominate the dataset.
* Customer ratings strongly influence sentiment classification.
* Visualizations help identify review patterns effectively.

---

## Project 3: Data Visualization

### Objective

Create meaningful visualizations from Netflix dataset using Python.

### Visualizations Created

* Movies vs TV Shows Comparison
* Content Distribution Pie Chart
* Netflix Release Trend Analysis
* Top 10 Countries by Content
* Content Added Per Year

### Outcome

Developed multiple visual representations to better understand trends and patterns in Netflix content.

# Netflix Data Analysis

## Project Overview
This project analyzes Netflix Movies and TV Shows data using Python. The objective is to explore trends in content releases, compare Movies and TV Shows, and identify countries contributing the most content on Netflix.

## Tools and Libraries
- Python
- Pandas
- Matplotlib

## Dataset
Netflix Movies and TV Shows Dataset from Kaggle.

## Analysis Performed
- Release trend analysis over the years
- Movies vs TV Shows distribution
- Top 10 countries with highest Netflix content

## Key Findings
- Netflix content increased significantly after 2015.
- Movies constitute a larger portion of the catalog than TV Shows.
- A small number of countries contribute a major share of Netflix content.
